How 1bucket works

In order to add something to a bucket, your tweets need to contain some key elements:

  1. @1b The tweet must begin with @1b.
  2. Bucket The first word after @1b defines the target bucket.
  3. Description Everything after the bucket name will appear as headline for your content.
  4. URL The content that you want to add to the bucket. 1bucket recognises a list of content types and automatically embeds them in the bucket.
